Tengo instalado json--server de forma global
npm install -g json-server
Uso: nos permite crear un servidor exponiendo una ruta a partir de un json en algún proyecto react 
Ejemplo:
Si yo tengo en mi proyecto productos.json con un array con objetos json de productos nos expone
este json como recurso según el nombre que le demos.
Ejecuto el comando: json-server --watch productos.json --port 5000
Y en mi navegador coloco: http://localhost:5000/productos     (el recurso tendrá el nombre del json)
Y de esta forma podemos simular una api para hacer funciones crud ya que productos.json se edita / agrega y elimian objetos
como una tabla de db

Atencion: productos.json actua como tabla de una base de datos y podemos modificarla a partir de que expone como recurso
en localhos:5000/productos gracias json-server instalado de forma global en mi mac
